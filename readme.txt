Converts Armenian characters in post slugs to Latin characters.

== Description ==

**This plugin is available for reference only. Its functionality has been replaced and extended by [Cyr-To-Lat](http://wordpress.org/extend/plugins/cyr2lat/).**

*The original author of this plugin is Anton Skorobogatov, who passed away prematurely in 2007.

Converts Armenian characters in post slugs to Latin characters. Very useful for Armenian-speaking users of WordPress.
You can use this plugin for creating human-readable links.

== Installation ==

1. Upload `hytolat` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
